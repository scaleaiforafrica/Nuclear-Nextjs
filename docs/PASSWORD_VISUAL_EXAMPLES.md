## Color Scheme

- **Weak:** Red (#EF4444) - bg-red-500, text-red-700
- **Fair:** Orange (#F97316) - bg-orange-500, text-orange-700
- **Good:** Yellow (#EAB308) - bg-yellow-500, text-yellow-700
- **Strong:** Green (#22C55E) - bg-green-500, text-green-700

- **Success:** Green (#F0FDF4) - bg-green-50, text-green-700
- **Error:** Red (#FEF2F2) - bg-red-50, text-red-600
- **Primary:** Purple (#9333EA) - from-purple-600

## Accessibility Features

1. **Keyboard Navigation:** All fields and buttons accessible via Tab key
2. **Screen Reader Support:** ARIA labels on all interactive elements
3. **Focus Indicators:** Visible focus rings (ring-2 ring-purple-500)
4. **Live Regions:** Password strength updates announced in real-time
5. **Touch Targets:** Minimum 44x44px on mobile for buttons
6. **Color Contrast:** All text meets WCAG 2.1 AA standards
